For any make and model without an FMS, the data should be recorded on a takeoff data card. This data is based on airplane weight, runway length available, runway gradient, field temperature, field barometric pressure, wind, icing conditions, and runway condition. Both pilots should review the takeoff data entered in an FMS or separately compute the takeoff data and cross- check with the takeoff data card. If takeoff plans change while taxiing, the pilot or crew should recalculate the takeoff data.
