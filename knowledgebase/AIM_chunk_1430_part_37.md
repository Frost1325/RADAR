AIRSPACE CONFLICT- Predicted conflict of an aircraft and active Special Activity Airspace (SAA).

AIRSPACE FLOW PROGRAM (AFP)- AFP is a Traffic Management (TM) process administered by the Air Traffic Control System Command Center (ATCSC) where aircraft are assigned an Expect Departure Clearance Time (EDCT) in order to manage capacity and demand for a specific area of the National Airspace System (NAS). The purpose of the program is to mitigate the effects of en route constraints. It is a flexible program and may be implemented in various forms depending upon the needs of the air traffic system.

AIRSPACE HIERARCHY- Within the airspace classes, there is a hierarchy and, in the event of an overlap of airspace: Class A preempts Class B, Class B preempts Class C, Class C preempts Class D, Class D preempts Class E, and Class E preempts Class G.
