 \(\mathrm{T - DP} =\) Temperature Dew Point Spread (TDS) \(\mathrm{TDS}\div \mathrm{CR} = \mathrm{X}\) \(\mathrm{X}\times 1,000\) feet \(=\) height of cloud base AGL

## Example:

Example: \(85^{\circ}\mathrm{F} - 71^{\circ}\mathrm{F} = 14^{\circ}\mathrm{F}\) \(14^{\circ}\mathrm{F}\div 4.4^{\circ}\mathrm{F} = 3.18\) \(3.18\times 1,000 = 3,180\) feet AGL The height of the cloud base is 3,180 feet AGL.
