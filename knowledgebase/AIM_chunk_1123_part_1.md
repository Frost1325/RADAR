d. Automated Surface Observing System (ASOS)/Automated Weather Sensor System (AWSS). The ASOS/AWSS is the primary surface weather observing system of the U.S. (See Key to Decode an ASOS/AWSS (METAR) Observation, FIG 7-1-9 and FIG 7-1-10. ) The program to install and operate these systems throughout the U.S. is a joint effort of the NWS, the FAA and the Department of Defense. AWSS is a follow-on program that provides identical data as ASOS. ASOS/AWSS is designed to support aviation operations and weather forecast activities. The ASOS/AWSS will provide continuous minute-by-minute observations and perform the basic observing functions necessary to generate an aviation routine weather report (METAR) and other aviation weather information.
