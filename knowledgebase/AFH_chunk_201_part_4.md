Once a straight- line ground reference is established, the pilot makes a \(360^{\circ}\) constant medium- banked turn. As the airplane completes the \(360^{\circ}\) turn, it should return directly over the straight- line ground reference but downwind from the starting point. Choosing a straight- line ground reference that has a crosswind, and using the same \(360^{\circ}\) constant medium- banked turn, demonstrates how the airplane drifts away from the reference even as the pilot holds a constant bank angle. In both examples, the path over the ground is not circular, although in reference to the air, the airplane flew a perfect continuous radius.
